# MatchaDex

Full-stack matcha cafe discovery platform with Google OAuth, interactive Mapbox search, and admin moderation dashboard.

Demo: https://matchadex.vercel.app

## Tech Stack
- Next.js
- TypeScript
- Supabase (Postgres + Auth)
- Prisma
- Mapbox
- TailwindCSS
- Vercel

## Features
- Google OAuth login
- Interactive cafe map
- Weighted ratings
- Favorites
- Admin moderation dashboard
